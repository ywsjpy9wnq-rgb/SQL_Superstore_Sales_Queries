# SQL_Superstore_Sales_Queries

## Summary
A collection of SQL queries demonstrating data analysis skills using the Superstore sales dataset.

## Dataset
- Source: Sample Superstore (public dataset)
- Contents: 9,994 orders, 3 categories, 4 regions

## Queries Included

1. **Regional Sales** - GROUP BY, SUM, AVG, COUNT
2. **Profit by Sub-Category** - Profit margin calculation
3. **Year-on-Year Growth** - CTE, LAG, YoY%
4. **Monthly Sales & Profit** - Date formatting, time trends
5. **Running Total Sales** - SUM OVER window function 
6. **Sales by Segment & Region** - Multi-column GROUP BY
7. **Top 3 Products per Category** - ROW_NUMBER, PARTITION BY
8. **Top 30 Customers** - CTE, filtering, LIMIT
9. **Day-over-Day Sales** - LAG, daily change
10. **Crosstab by Region** - CASE WHEN conditional aggregation
11. **RFM Analysis** - NTILE, customer segmentation
12. **7-Day Moving Average** - ROWS BETWEEN, trend smoothing

## How to Use

1. Load the Superstore dataset into any SQL database
2. Run the queries in order
